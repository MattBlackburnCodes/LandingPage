About

Hello! I'm Matthew Blackburn, an experienced IT professional transitioning into frontend web development. My background includes roles like Senior End User Analyst and Desktop Support Specialist, where I honed my skills in system administration, network management, and customer support. Now, as I shift towards web development, I'm leveraging my expertise in complex tech environments, problem-solving, and software/hardware integration.

Currently deepening my knowledge in HTML, CSS, JavaScript, Python, and more, I bring a unique blend of administrative and creative insights to my projects. I hold a BS in Computer Science from Colorado Technical University and actively enhance my skills through platforms like The Odin Project and W3 Schools. Passionate about creating user-friendly web experiences, I am excited to apply my diverse skills in this new realm and contribute to innovative web development teams.

Blue Bicho Studio

As the founder of a game studio, I have successfully developed and published several games using Unity and Visual Studio 2022. My creations are showcased on https://blue-bicho.itch.io/, featuring a diverse range of engaging titles:

1. Papcorn the Pirate on Skull Island: This game is an exciting point-and-shoot side scroller, where players embark on an adventurous journey with Papcorn the Pirate, navigating through the perilous Skull Island.

2. Blastroids:Set in the vast expanse of space, Blastroids is a dynamic space shooter game that challenges players to navigate through asteroid fields and engage in intense combat.

3. Mako II Enter the Deep:This side scroller game offers an underwater adventure, where players control a shark exploring the depths of the ocean, encountering various marine life and obstacles.

The Great Binary Duel: Humanity's Last Throw

In this project, I created a web based engaging Rock Paper Scissors game called The Great Binary Duel: Humanity's Last Throw, accessible at https://rock-paper-scissors-smoky-rho.vercel.app/match/match.html. Developed using Visual Studio Code, the game combines HTML, CSS, and JavaScript to offer a seamless and interactive user experience. I enriched the game's appeal by integrating custom art designed with Midjourney, adding a unique visual touch. Furthermore, ChatGPT was utilized to weave an intriguing story into the gameplay, providing an immersive narrative dimension. This project not only highlights my technical skills in web development but also showcases my ability to creatively blend programming with storytelling and art design.
